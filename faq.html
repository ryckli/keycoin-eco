常见问题

1. Key Coin 和比特币有什么区别？

比特币锚定稀缺性（2100 万上限），Key Coin 锚定实际发电量（每日动态调整）。比特币是「数字黄金」，Key Coin 是「数字电力」。

2. Key Coin 怎么抵消电费？

通过集成钱包直接抵扣，1 KEY ≈ 1 kWh 等价价值。覆盖 AI 算力约 70% 核心成本。与电网/能源企业 API 打通后实现自动扣减。

3. 如何获得 Key Coin？

三种方式：① 建设清洁能源设施发电获取；② 创作内容获 AI 评估奖励；③ 在二级市场购买（Uniswap LP 即将上线）。

4. 谁控制 Key Coin？

9 人理事会（长期持有者选举）+ 所有锁定 ≥2 年的持有者拥有投票权。没有中心化团队独裁。理事会年度改选 1/3。

5. 如果量子计算机破解了区块链怎么办？

Key Coin 已部署 QuantumMigrationCommitment 合约（0x186a...0DFc），采用五级威胁响应机制：

  • 当前威胁等级：CONCERNED（2 级） — 已进入混合签名模式准备阶段
  • 采用 Hybrid AND 模式：ECDSA + ML-DSA(Dilithium) 双重验证，杜绝降级攻击
  • 威胁升级到 CRITICAL 时自动冻结非 PQC 高价值交易，90 天强制迁移
  • 最终迁移目标：CRYSTALS-Dilithium (FIPS 204)，对齐 2035 全球 PQC 迁移截止线

详见技术架构页 → 后量子密码学（PQC）技术架构

2026 Q2 基础代币已上线以太坊主网。4 合约开源验证。9 人理事会就位。Oracle 网络运行中。正在筹备 Uniswap LP 和 Etherscan Token Update。
